# GuardianWill — Effortless Estate Planning

GuardianWill is a modern, professional web application designed to simplify estate planning for families. Our platform allows users to draft essential legal documents like Wills and Estate Administration plans with ease, security, and affordability.


## 🛡️ Key Features

- **Intuitive Questionnaire:** A user-friendly interface that translates complex legal requirements into simple language.
- **Modern UI/UX:** A premium, responsive design with smooth scroll animations and clear "Core Philosophy" pillars.
- **Direct Support:** Integrated WhatsApp "Click to Chat" buttons for immediate assistance with professional advisors.
- **Secure Data Handling:** Contact form submissions are stored securely in a dedicated SQLite3 database for administrative review.
- **Professional Layouts:** Clean, card-based service descriptions and interactive statistics.

## 🛠️ Technology Stack

- **Backend:** Django (Python)
- **Frontend:** HTML5, Vanilla CSS, JavaScript
- **Database:** SQLite3
- **Animations:** Custom Intersection Observer API
- **Branding:** Citadel-inspired theme (Navy & Cyan)

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Django 4.0+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/guardianwill.git
   ```

2. Navigate to the project directory:
   ```bash
   cd guardianwill
   ```

3. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

5. Run migrations:
   ```bash
   python manage.py migrate
   ```

6. Start the development server:
   ```bash
   python manage.py runserver
   ```

7. Access the site at `http://127.0.0.1:8000`.
